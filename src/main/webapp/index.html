Bienvenue sur Mediashare!

Le projet mediashare est ne lors d'un projet collaboratif de sept etudiants, au sein de l'IUT A de l'universite de Lille1
Il est heberge sur un serveur tomcat et utilise les technologies suivantes:
Java, Javascript, html, css, REST, sql pour la base de donnees
Le but est de pouvoir partager et telecharger des fichiers en tout genre, mais aussi de pouvoir tchater avec les autres utilisateurs, de gerer une liste d'amis et de commenter les uploads.
